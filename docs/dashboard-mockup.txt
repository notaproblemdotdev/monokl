## Key Legend

| Key | Action |
|-----|--------|
| Tab | Switch between sections |
| j / ↓ | Move down |
| k / ↑ | Move up |
| o | Open selected item in browser |
| q | Quit application |
| Enter | Show item details (future) |

## Color Scheme (if terminal supports it)

- **Active section border**: Blue accent
- **Inactive section border**: Gray/dim
- **High priority items**: Red indicator (!)
- **Medium priority**: Yellow indicator (#)
- **Low priority**: Gray indicator ( )
- **Status badges**: Cyan/blue
- **Key hints**: Dimmed text at bottom
